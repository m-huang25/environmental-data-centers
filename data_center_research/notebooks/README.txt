Notebooks Folder

This folder contains Jupyter notebooks used for exploration, analysis, and visualization throughout the project. Notebooks provide step-by-step workflows with narrative explanations, including:

Exploratory Data Analysis (EDA)
Data cleaning walkthroughs
Spatial analysis
Visualizations with commentary

Organization
Current/Main Notebook: The main, finalized notebook that reflects the complete analysis is stored directly in this folder.
Archived Notebooks: All previous drafts, exploratory versions, or intermediate notebooks are stored in the archive/ subfolder. These are kept for reference but are not part of the primary workflow.

Notes
Only the main notebook is intended for active use and sharing.
Archived notebooks can provide context, historical analysis, or alternative approaches but are not required to run the project workflow.
